Introduction

This introductory documentation Raspian Operating System for Raspberry Pi.
Raspbian is the recommended operating system for Raspberry Pi (being optimized for its hardware) and is based on a GNU/Linux distribution Debian.

Raspbian was created by Mike Thompson and Peter Green as an independent project1. The initial launch was in June 2012. The operating system is still in active development. Raspbian is highly optimized for the Raspberry Pi.

What is Raspbian

Raspbian is an unofficial port of Debian (GNU/Linux) wheezy armhf with compilation settings adjusted to produce code that uses "hardware floating point", the "hard float" ABI and will run on the Raspberry Pi.
The port is necessary because the official Debian wheezy armhf release is compatible only with versions of the ARM architecture later than the one used on the Raspberry Pi (ARMv7-A CPUs and higher, vs the Raspberry Pi's ARMv6 CPU).

The Debian squeeze image issued by the Raspberry Pi foundation was based on debian armel which uses software floating point and the "soft float" ABI. The foundation used the existing Debian port for less capable ARM devices. Therefore, it did not use of the Pi's processor's floating point hardware - reducing the Pi's performance during floating point intensive applications - or the advanced instructions of the ARMv6 CPUs

What should I know

The goal of Raspbian is to become the leading OS of choice for all users of the Raspberry Pi.
Raspbian images are produced by various people. For newcomers we recommend the images provided by the Raspberry Pi foundation available in their download page.

Raspbian tries to stay as close to Debian as reasonably possible. Any information you find that applies to Debian will almost certainly apply to the same version of Raspbian. Information for earlier versions of debian will often apply too but may need some tweaks just as it would when using it on a newer version of debian.

How to Install

To install Raspbian on our Raspberry Pi we have two versions; one more complete with graphical environment and a smaller without graphical environment:

  • Raspbian Pixel:

    Pi Improved X-Window Environment, Lightweigh is the full version with Raspbian graphical environment, i.e. the desktop version with menus, windows, icons, wallpapers, etc. used by most users as a desktop computer.

  • Raspbian Lite:

    Version without graphical environment, i.e. the console mode version without graphs. This option is generally for advanced users with knowledge of Linux who use the Raspberry Pi as a server.

In order to install the Raspbian operating system, we must first access the downloads section from The Raspberry Pi Foundation website and choose from two options for installation.

  • Via NOOBS
  • Through an OS image
What is NOOBS

NOOBS or "New Out Of Box Software" is a simple step-by-step guided installer, this option includes a quick edit method of configuring the new system, a pre-installed web browser and a command line recovery mode from a rescue partition that is automatically created.

You can choose between
- NOOBS Pixel: Full version with graphical environment (windows, folders, etc ; uses about 2G of space).
- NOOBS Lite: Reduced version without graphic environment (console mode, uses about 32Mb of space).

Install NOOBS

Before installing Raspbian we have to format the SD card with FAT32 format. To do this we can download the free SDCard software, or format it using the command line.

diskutil listsudo diskutil eraseDisk FAT32 SDCARD MBRFormat /dev/disk2

Next we copy the downloaded NOOBS files inside the card.

Last connect your Raspberry Pi and select the options offered by the installation wizard.
The installation process takes about 15-20 minutes.

Download Raspbian Image

An image is a file containing the structure and complete contents of an operating system.
This is useful when we want all our computers to have the same configuration.
We can create a default image and save it as a backup, saving the time when we need to re-install and configure our Raspberry Pi.

The Raspbian with Desktop image contained in the ZIP archive is over 4GB in size, these archives use features not supported by older unzip tools on some platforms. We recommend using 7Zip (Windows) or The Unarchiver (Macintosh). Both are free and have been tested to unzip the image correctly.

You can Choose between this official Versions on The Raspberry Pi Site

  • Raspbian Buster with desktop and recommended software
  • Raspbian Buster with desktop
  • Raspbian Buster Lite

There are no huge differences between Debian Stretch and Debian Buster, most of the differences are security changes designed to make Buster harder to hack. If you want to install previous versions of Raspian (like Jessie or Stretch) you can visit This Site

Once we donwload our Image Version, we have to pass the content of the Raspbian image to our microSD. For this we can download the free Etcher software, or add the operating system image using the command line. This process usually takes about 20 minutes.

Note: the use of the dd tool can overwrite any partition of your machine. If you specify the wrong device in the instructions below, you could delete your primary Mac OS partition. Please be careful.

Open a terminal and run:

diskutil list

Identify the disk (not the partition) of your SD card,

Unmount your SD card by using the disk identifier, to prepare it for copying data
Where "disk" is your BSD name e.g

diskutil unmountDisk /dev/disk

Copy the data to your SD card:

sudo dd bs=1m if=image.img of=/dev/rdisk conv=sync

This may result in a dd: "invalid number '1m' error" if you have GNU coreutils installed. In that case, you need to use a block size of 1M in the bs= section, as follows:

sudo dd bs=1M if=image.img of=/dev/rdisk conv=sync

If this command still fails, try using disk instead of rdisk, for example:

sudo dd bs=1M if=2019-07-10-raspbian-buster.img of=/dev/disk4 conv=sync

Eject the card:

sudo diskutil eject /dev/rdisk

Finally, we will connect our Raspberry Pi to prove that Raspbian has been installed correctly.

Keep Raspbian Update

Once we have installed Raspbian on our Raspberry Pi, either by following the NOOBS wizard or by installing an image, it is convenient to update.
To do this, access the terminal and execute the following commands:

pi@raspberrypi: ~ $ sudo apt-get update

With this command, we are updating the repositories

pi@raspberrypi: ~ $ sudo apt-get upgrade

With this command we do an update of our operating system and the applications that are contained in the repositories.
